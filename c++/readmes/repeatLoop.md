

# While
usamos o while quando queremos checar se algo é verdadeiro, e deve repetir algo.
usamos com a variavel "i" para incrementar até a verdade se tornar falsa.

while(condição){
    ação
}

int n = 4
int i = 0

while(i <>= n){
    "ação"
    i++
}
